1.	В чем разница между innerHTML и textContent и когда следует использовать каждый метод?
innerHTML - парсит и вставляет HTML-разметку, может выполнять скрипты (риск XSS)
textContent - работает только с текстом, безопаснее и быстрее
Использовать: textContent для текста, innerHTML только когда нужна разметка от доверенного источника

2.	Как работает делегирование событий (event delegation) и какие преимущества оно дает при работе с динамическими элементами?
Принцип: Обработчик вешается на родителя, события обрабатываются по всплытию
Преимущества: Работает с динамическими элементами, меньше обработчиков, лучше производительность

3.	Какие методы используются для поиска элементов в DOM и в чем разница между getElementById, querySelector и querySelectorAll?
getElementById - ищет по ID, возвращает один элемент
querySelector - ищет по CSS-селектору, возвращает первый элемент
querySelectorAll - ищет по CSS-селектору, возвращает NodeList
Разница: querySelector* более гибкие, но медленнее для простых случаев

4.	Как предотвратить стандартное поведение элемента при обработке событий и когда это необходимо?
Метод: event.preventDefault()
Когда нужно: При кастомной обработке форм, ссылок, драг-энд-дроп

5.	В чем разница между classList.add() и className для работы с CSS классами?
classList - точечное управление классами (add/remove/toggle)
className - полная перезапись всех классов
Преимущество classList: Не затрагивает другие классы, удобнее для манипуляций

6.	Как создать новый элемент и добавить его в DOM несколькими способами? Какой способ наиболее эффективен?
Способы: createElement+append, innerHTML, insertAdjacentHTML
Самый эффективный: createElement для сложных структур, insertAdjacentHTML для простых

7.	Что такое всплытие событий (event bubbling) и как работает метод stopPropagation()?
Всплытие: Событие поднимается от целевого элемента к корню
stopPropagation() - останавливает всплытие, но не предотвращает действие
Использовать осторожно: может нарушить другие обработчики

8.	Как получить данные из формы и в чем разница между методами сбора form data?
Способы: FormData API, прямой доступ к элементам, сериализация
FormData преимущества: Автоматическая сборка, поддержка файлов, удобно для отправки

9.	Какие события используются для обработки ввода данных и в чем разница между input, change и keyup?
input - при любом изменении (реальное время)
change - после потери фокуса и изменения
keyup - при отпускании клавиши
Выбор: input для мгновенной реакции, change для финального значения

10.	Как оптимизировать производительность при частых манипуляциях с DOM?
Использование DocumentFragment для групповых операций
Минимизация reflow (изменение стилей батчами)
Кеширование ссылок на элементы
Виртуализация для больших списков
Дебаунсинг частых операций
