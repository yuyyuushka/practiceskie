# Task Manager
Full-stack приложение для управления задачами с React и Express

## Структура проекта

### Backend (`/backend`)
- `server.js` - Express сервер с API
- `package.json` - зависимости бэкенда  
- `test-items.js` - тесты API

### Frontend (`/frontend`)
- `App.jsx` - главный компонент
- `components/` - UI компоненты
- `hooks/useItems.js` - управление состоянием
- `services/api.js` - HTTP клиент
- `package.json` - зависимости фронтенда

## Запуск

## Backend
cd backend
npm install
node server.js
